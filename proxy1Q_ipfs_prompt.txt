4. FUNCTION SPECIFICATION
extract_host(data: bytes) -> str
Input: Raw bytes of the HTTP request
Logic: Decodes to ASCII, ignoring errors. Searches for CONNECT <host> or the Host: <host> header using re.search. Returns the found host or "unknown" on error/no match.
Output: str
encode_data(data: bytes) -> bytes
Input: Raw data to send
Logic: Converts data to HEX using binascii.hexlify(), appends \n. Generates a frame: HTTP_PREFIX + SEPARATOR + hex_data.
Output: bytes
decode_data(data: bytes) -> bytes
Input: Frame with prefix and hex strings separated by \n
Logic: If it starts with HTTP_PREFIX, truncates it and the SEPARATOR. Splits the remainder on \n, removes empty lines, and decodes each string from hex using binascii.unhexlify(). Concatenates the result. Decoding errors are ignored.
Output: bytes
forward_to_tunnel(tunnel_sock: socket, client_sock: socket) -> None
Direction: Client → Tunnel
Logic: In a loop, reads up to 8192 bytes from client_sock, encodes using encode_data(), and sends to tunnel_sock. Aborts upon receiving empty data. Wrapped in try/except.
forward_to_client(tunnel_sock: socket, client_sock: socket) -> None
Direction: Tunnel → Client
Logic: Reads chunks from tunnel_sock and stores them in buffer. If \n is present, divides the buffer into lines and passes each line (appended with \n) to decode_data(). If decoding is successful, sends the result to client_sock. Works in a loop.
handle_browser(client_sock: socket, client_addr: tuple) -> None
Role: Handler for a single client connection
Logic:
Reads the initial request, retrieves the host.
Finds a free local port via a temporary socket (bind(127.0.0.1, 0)).
Starts subprocess.Popen(['ipfs', 'p2p', 'forward', PROTOCOL, f'/ip4/127.0.0.1/tcp/{local_port}', f'/p2p/{SERVER_PEER_ID}'], stdout=DEVNULL, stderr=DEVNULL).
Waits 2 seconds (time.sleep(2)).
Connects to 127.0.0.1:local_port, sets TCP_NODELAY.
Sends an encoded initial request.
Starts two threads (forward_to_tunnel, forward_to_client) with daemon=True.
Blocks on join().
In finally: closes the tunnel socket, terminates the subprocess, closes the client socket. main() -> None
Logic: Creates a server socket, sets SO_REUSEADDR, binds to LISTEN_IP:LISTEN_PORT, and listens on a queue of up to 100 connections. Accepts connections in an infinite loop, sets TCP_NODELAY, and starts handle_browser in a daemon thread. Handles KeyboardInterrupt for graceful shutdown.
